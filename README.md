# Gerenciador de Tarefas
Implementação de CRUD para gerenciamento de tarefas.

## Operações

- Cadastro de usuário
- Login
- Cadastro de Tarefa
- Edição de Tarefa
- Exclusão de Tarefa
- Listagem de Tarefas Pendentes
- Listagem de Tarefas Finalizadas

## Tecnologias

- Java 8
- TomCat 8.5.65
- JSP 2.2
- JPA 2.1
- Postgresql 13
- Docker
